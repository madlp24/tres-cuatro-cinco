# tres-cuatro-cinco

Primera base del nuevo sitio para `Tres Cuatro Cinco steakhouse`.

## Archivos principales

- `index.html`: home inicial inspirada en la referencia visual y ajustada al brand book.
- `styles.css`: sistema visual responsive con paleta negra, roja y gris clara.
- `script.js`: menu movil, enlace unico de reservas y feedback del newsletter.

## Conexiones activas

- Reservas reales conectadas a `https://menupp.co/trescuatrocinco/booking/venue/MrO2svW7FCkLsW7eEXes`.
- Formulario de newsletter listo para integrarse con el proveedor que elijan.

## Deployment recomendado

Para esta primera etapa recomiendo `GitHub Pages` porque:

- el proyecto es estatico;
- ya esta ligado a GitHub;
- es facil de publicar y mantener;
- despues podemos conectar dominio propio desde GoDaddy.

## Siguiente paso

- Subir la carpeta de fotografias al repo para reemplazar los placeholders.
- Conectar el formulario a Mailchimp, Brevo o el servicio elegido.
- Publicar el sitio en GitHub Pages y luego apuntar el dominio.
